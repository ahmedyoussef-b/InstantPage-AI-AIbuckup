# AGENTIC - Assistant IA Professionnel Hybrid RAG

AGENTIC est un assistant intelligent de nouvelle génération conçu pour une exploitation technique et industrielle. Il repose sur une architecture **100% locale** garantissant une confidentialité totale des données et une performance d'élite grâce à 15 innovations majeures.

## 🚀 Les 15 Innovations Majeures

### 1. Architecture Multi-Modèles & Routeur Sémantique
Le système dirige chaque question vers le modèle le plus adapté (Phi-3 pour la technique, Llama-3 pour le légal, etc.) pour une précision maximale.

### 2. Cache Sémantique Intelligent
Réduction drastique de la latence grâce à la détection sémantique des questions similaires, évitant les appels redondants au LLM.

### 3. RAG Hybride (Vectoriel + Graphe + Temporel)
Fusion de trois sources de contexte : recherche vectorielle pour le sens, graphe pour les relations, et temporelle pour la fraîcheur des données.

### 4. Knowledge Graph Local Automatique
Extraction automatique des relations thématiques pour construire un graphe de connaissances local persistant sans intervention humaine.

### 5. Prompt Engineering Dynamique Adaptatif
Ajustement en temps réel des instructions système selon l'intention détectée (procédure, définition, analyse comparative).

### 6. Chaîne de Pensée (CoT) Dynamique
Décomposition automatique des problèmes complexes en étapes de réflexion logiques dont la profondeur s'adapte à la difficulté.

### 7. Support de Modèles Quantifiés Spécialisés
Optimisé pour le chargement de modèles ultra-légers (q4_k_m, ~350MB) via Ollama, spécialisés sur vos documents propres.

### 8. Apprentissage Continu Ultra-Léger
L'IA apprend de vos corrections manuelles pour s'auto-corriger localement sans ré-entraînement lourd du modèle.

### 9. Raisonnement par Contraste (Contrastive Reasoning)
Clarification des concepts complexes en explorant systématiquement ce qu'ils ne sont pas par rapport aux alternatives proches.

### 10. Vérification Auto-Consistante (Self-Consistency)
Résistance aux hallucinations par génération de multiples chemins de raisonnement indépendants et vote majoritaire sur la conclusion.

### 11. Arbre de Décision Latent (Latent Decision Tree)
Exploration exhaustive des bifurcations logiques pour les choix stratégiques complexes, garantissant qu'aucune option n'est ignorée.

### 12. Raisonnement Analogique (Analogical Reasoning)
Transfert de solutions basées sur l'expérience passée et les problèmes similaires résolus avec succès, simulant l'expertise métier.

### 13. Méta-cognition (Self-Awareness)
L'IA évalue sa propre compréhension et confiance avant de répondre. Elle admet ses limites si l'information est insuffisante.

### 14. Raisonnement Contrefactuel (Counterfactual Reasoning)
Exploration des scénarios "et si" pour identifier les causes racines et comprendre les relations causales profondes derrière les événements techniques.

### 15. Raisonnement Modulaire (Modular Reasoning)
Décomposition des questions multi-facettes en modules spécialisés (Logique, Causal, Mathématique) pour une expertise intégrée à 360°.

## 🛠️ Stack Technique

- **Frontend** : Next.js 15 (App Router), React 19, Tailwind CSS.
- **AI Engine** : Genkit 1.x, Ollama (Modèles locaux).
- **Storage** : Virtual File System (VFS) persistant atomique via LocalStorage.
- **Voice** : STT (WebSpeech) & TTS (Piper/Edge local).
